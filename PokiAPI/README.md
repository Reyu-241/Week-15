# All Generation 1 Pokémon 
Pokédex of all 151 Pokémon randomly placed in each province of South Africa.

# Pokémon cards:
*   **name**: *Pokémon name*
*   **region**: *The region/province of SA a Pokémon is Located*
*   **img**: *URL to an image of this Pokémon*
*   **moves**: *The number of moves a Pokémon can use*
*   **type**: *Pokémon type*
*   **weakness**: *Types of Pokémon this Pokémon is weak to*

# Pokémon Team:
*   **slots**: *6 slots for a team of Pokémon*
*   **stats**: *Very basic stats layout for a Pokémon team*
