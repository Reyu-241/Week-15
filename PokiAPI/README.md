# All Pokémon from Generation 1 to Generation 3
Pokédex of all 386 Pokémon randomly placed in each province of South Africa.

# Pokémon cards (front):
*   **name**: *Pokémon name*
*  **species**: *The type of species a Pokémon is*
*   **region**: *The region/province of SA a Pokémon is Located*
*   **img**: *URL to an image of this Pokémon*
*   **moves**: *The number of moves a Pokémon can use*
*   **type**: *Pokémon type*
*   **weakness**: *Types of Pokémon this Pokémon is weak to*

# Pokémon cards (back):
* **desription**: *A simple description of the Pokémon*
* **base**: *The base stats of a Pokémon*

# Pokémon Team:
*   **slots**: *6 slots for a team of Pokémon*
*   **stats**: *Very basic stats layout for a Pokémon team*
